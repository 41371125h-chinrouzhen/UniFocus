# Programming-Language-Final

##專題提案—— UniFocus：台師大智慧學習導航員 [影片鏈接](https://youtu.be/aZ41LtePE0E)

UniFocus 是一個專為台師大學生打造的智慧學習 App。解決大學生「課表美感不足」與「學習資訊斷層」的痛點，將單純的時間管理工具，升級為全方位的學習導航系統。

✅ 智慧課表 (Smart Schedule)： 自動解析 PDF 課表，一鍵生成美感介面。 

✅ 動態導航 (Active Navigation)： 串接全校課程資料庫，自動推播當週進度與預習影片。 

✅ 知識地圖 (Mind Map)： 生成課後思維導圖模板，協助知識固化。
